# 经济模拟器 v1.0 — 猫步天下

自动化蒙特卡洛模拟验证 v1.0 全部货币与资源闭环。

## 一键运行

```bash
cd tools/econ_sim

# 运行单个 profile
python run_sim.py --profile medium_ad_off_nocard --days 30 --iterations 100

# 运行全部 12 个 profile × 30/90 天
python run_sim.py --all --iterations 100
```

## 输出目录

- `output/{profile_name}_30d.csv` — 日粒度曲线（P50/P95）
- `output/{profile_name}_90d.csv` — 90天粒度曲线
- `output/assertion_report.md` — A1~A10 断言报告

## 参数配置

所有数值从 `config/params.json` 读取，键名与 GDD v3.1 §18.1 速查表及各章参数表一一对应。

修改参数后重跑即可回归验证。

## GDD 设计意图 vs 模拟实测差异说明

### 广告能量占比（A3）
- **GDD 假设**：广告能量 ≤ 50% 总能量
- **模拟实测**：低活跃+广告全开时广告占比 90.8%，中活跃 65%，高活跃 49%
- **根因**：低步数用户日能量产出极低（800），广告提供 9000 能量 → 广告绝对主导
- **建议**：若需保护"步数驱动"支柱，降低广告位#1 日上限或单次奖励量

### 中活跃橘猫满级（A4）
- **GDD 假设**：30±5 天
- **模拟实测**：中活跃 30 天 ✅，高活跃仅 19 天 ❌
- **根因**：高活跃日步数 8000 → 携带猫日经验 9200 → 150K 总经验在约 16 天达标

### 孵化拓展达日（A2）
- **GDD 假设**：中活跃 D20~D40 达成第一档扩容（5000 金币）
- **模拟实测**：中活跃 D6.6、高活跃 D4.0、低活跃 D23~34
- **根因**：高能量产出 → 快速孵化 → 快速积累 pokedex → 提前触发扩容

### 能量截断（A6）
- **GDD 假设**：中活跃日截断=0，高活跃 ≤1 次/周
- **模拟实测**：无广告时 ✅，有广告时 ❌（中活跃 90 天累计截断 ~145K 能量）
- **根因**：广告 9000 + 步数 4700 = 13700/天，消耗仅 4250/蛋 → 急剧溢出
