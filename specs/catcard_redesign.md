# CatCard 重设计 SPEC

## 布局（720x1280 设计尺寸）

整个 CatCard 占屏幕下半屏：

```
┌──────────────────────────────────────┐  y=480
│          [半透明遮罩 - 点击关闭]      │
├──────────────────────────────────────┤  y=640
│                                      │
│          [ 猫咪展示区域 ]            │  y=640~900
│        (TextureRect 显示猫帧)        │     260px
│                                      │
├──────────────────────────────────────┤  y=900
│  [名字]    [品种·稀有度]            │     40px info row
├──────────────────────────────────────┤  y=940
│  [🍖喂食]  [✋抚摸]  [🎾玩耍]       │     56px button row
│         [🧭探索(左)]                │
├──────────────────────────────────────┤  y=996
│  [状态文字: 冷却中/可互动/..]       │     30px status
├──────────────────────────────────────┤  y=1026
│     [提示:点击空白处关闭]            │     24px hint
├──────────────────────────────────────┤  y=1050
│    底部留白到底                    │
└──────────────────────────────────────┘  y=1280
```

## 猫咪展示（关键功能）

### 显示
- 用 TextureRect 显示猫咪当前品种的正面帧（front_right/front_left）
- 从 CatSprite 的帧加载逻辑复用：使用品种名加载对应帧图

### 交互动画
- **喂食(🍖)**: 猫咪低头吃 → 抬头舔嘴 → 循环3次
- **抚摸(✋)**: 猫咪眯眼 → 蹭手 → 恢复
- **玩耍(🎾)**: 猫咪跳跃 → 转身 → 扑
- 每个动画用 3-5 帧切换，timer 控制节奏

## 交互按钮
- 喂食/抚摸/玩耍：调用 InteractionSystem 已有逻辑
- 探索：保持不变（弹探索时长选择）
- 所有按钮带冷却状态显示

## CatCard.gd 修改

### 新增成员
- `_cat_display: TextureRect` - 猫咪展示区域
- `_frames_cache: Dictionary` - 缓存加载的帧图
- `_anim_timer: Timer` - 动画切换计时器
- `_anim_frames: Array[Texture2D]` - 当前动画帧列表
- `_anim_index: int` - 当前帧索引

### 新增方法
- `_load_cat_display(cat_data)` → 加载猫品种帧图到展示区
- `_play_anim(action_name: String)` → 播放指定动画
  - 动作: "idle" / "eating" / "petting" / "playing"
  - 每帧 0.15-0.25s，循环播放
- `_stop_anim()` → 停止动画，回到 idle 帧

### 其他不变
- 交互逻辑（_do_interaction / _on_feed_pressed 等）
- 冷却管理
- 探索/送养逻辑
- 打开/关闭动画
